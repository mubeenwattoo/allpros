| `why-google.webp` | Feature card — Google / search visibility |
| `why-ai.webp` | Feature card — AI recommendations |

| `how-step-2.webp` | How it works — step 2 illustration |

If you only have **PNG**, export as PNG and either **rename** to the `.webp` paths above and change extensions in `index.html`, or replace `.webp` with `.png` in each `<img src="...">`.

Recommended width for hero mockup: **1200–1600px**. Card visuals: **800–1200px** wide.
